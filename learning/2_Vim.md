### VIM

|keyWord|Description|
|:--:|--|
|a|Add Character|
|i|Insert Character|
|Shift + a | Add Character to EndLine|
|Shift + i | Insert Chracter to StartLine|
|h(←), j(↓), k(↑), l(→) | Move Cursor |
| W | Move front Word|
| b | Move Back Word
| Shift + h | Move Top Screen |
| Shift + l | Move Bottom Screen |
| [[ | Move First Line |
| ]] | Move Last Line | 
| { | Move Bottom paragraph|
| } | Move Top paragraph|
| v | Select Character |
|Shift + v | Select Line |
| y | Copy |
| p | Paste |
| d | Delete Character|
| dd | Delete Line|
| Shift + > | Indent |
| Shift + < | Outdent |
| x | Delete Character |
| r | Modify Character | 
| Ctrl + u | Page Up |
| Ctrl + d | Page Down |
